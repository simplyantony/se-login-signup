se-login-signup is a small, static login and signup page built using HTML and CSS. The repository contains two pages-Login_Setup.html for signing in with a user id and a password, and Signup_Setup.html for creating an account with first name, last name, user id, and password, all while sharing a stylesheet--styles.css-- that both pages load for consistent styling.

To us the project, you don't need any build tools or dependencies; all you need is to download or clone the repo and open either Login_Setup.html or Signup.html directly in your browser, since both pages link to one another. 

The two pages are connected with simple relative links, so the login page navigate to the signup page via the file name and vice versa. Both pages reference styles.css using a standard link tag.

At the moment, this is only a simple frontend integration. If you want to customize the project, feel free to do so. This is an unlicensed repository, so if you plan to reuse or distribute the code, I recommend that you add an explicit license.
